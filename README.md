
# Scan-Zxing使用方法
可以生成二维码，扫描二维码和相册中的二维码


1. Add it in your root build.gradle at the end of repositories:
在project的build.gradle添加如下代码：	

		allprojects {
			repositories {
				maven { url 'https://jitpack.io' }
			}
		}

2. Add the dependency
在build.gradle添加依赖：	

		dependencies {
			compile 'com.github.goodboy321:Scan-Zxing:1.0'
		}


具体需求可修改源码
